## 安装

使用pip直接安装whl即可

> 可能会出现权限相关的问题，管理员安装即可，待修正

## 使用

第一次运行时要求输入`API key`，会保存到`config.txt`文件中，该文件位于`User\[username]\Jarivs_log\`中


正常使用：输入，按两次回车发送。

## 程序参数

- 显示所有命令 : `/list`, `/?`, `/help`
- 显示所有角色 : `/listR`, `/cr`
- 改变角色: `/change_role`, `/cr`
- 显示历史: `/hisuse`, `/hu`，包含月使用量、余额、用户限额。
- 清屏: `/cls`
