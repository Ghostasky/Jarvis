## 安装

直接pyinstaller打包放到任意盘的Jarvis目录，将`Jarvis.exe`所在目录添加到环境变量即可。

## 使用

第一次运行时要求输入`API key`，会保存到`config.txt`文件中。

正常使用：输入，按两次回车发送。
